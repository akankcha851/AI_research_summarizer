# 📄 AI Research Paper Summarizer

A simple web app built with Streamlit that lets users upload research PDFs and get quick AI-generated summaries using Hugging Face.

---

## 🚀 Features
- Upload PDF files  
- Extract text automatically  
- Generate AI summaries  
- Simple and clean UI  
- Download summary  

---

## 🛠️ Tech Stack
- Streamlit  
- Hugging Face API  
- PyMuPDF (fitz)  
- CSS  
- python-dotenv
